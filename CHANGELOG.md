# Changelog

All notable changes to CloakCraft will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Core privacy transfer circuits (1x2, 2x2)
- Shield/unshield functionality
- AMM swap circuits and instructions
- Order book circuits for private limit orders
- Governance voting circuits with threshold decryption
- Note scanning and balance tracking
- React UI components library
- TypeScript SDK with full API coverage

### Security
- Groth16 proofs for all private operations
- BabyJubJub curve for stealth addresses
- Poseidon hash for commitments
- Light Protocol integration for compressed state

## [0.1.0] - 2024-XX-XX

### Added
- Initial release
- Private token transfers
- Basic shield/unshield operations
- SDK alpha
